# Smart-Motion-Controlled-Fan-System
A motion-controlled fan prototype designed to improve user comfort by automatically directing airflow toward a detected user. This project was developed as a first embedded systems build during secondary school, focusing on basic automation, sensor input, and user interaction.

# Overview
The system uses a motion sensor to detect the presence and position of a user within a room. When motion is detected, the fan automatically adjusts its direction by stopping its oscillation toward the user, ensuring airflow is directed at them without requiring manual adjustment.

To improve usability and awareness in shared spaces, an LED indicator is activated when the fan is locked onto a user’s position. This signals to others in the room that the system has responded to someone’s presence.

## Features
- Motion-based user detection
- Automatic fan direction control (oscillation stop mechanism)
- User-targeted airflow adjustment
- LED indicator to show active user targeting
- Shared-space awareness functionality

## Components & Tools
- Motion sensor
- Servo motor
- LED indicator
- Breadboard and jumper wires
- Arduino
